# My Online Clinic App

<p>This application is a client-server API built using Mongo DB and Node.js in the backend and React in the fron end. Mongoose library was utilized to interact with the database and express framework to develop server functionality. WebSockets were utilized to implement the Chat feature once the user is authenticated. Database used is cloud hosted Atlas Mongo DB and both server and client are deployed successfully at Heroku 
https://samer-online-clinic.herokuapp.com/login</p>
